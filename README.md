# Верстка сайта интернет-магазина 3elegant

HTML, SCSS, JS, GULP

### В проекте реализовано:
- Закрытие верхнего рекламного блока при помощи js
- Кастомный swiper для блока NewArrivals
- Таймер на js
- Карточки товара выполнены через шаблон json
